# Smart Home Dashboard App

A modern IoT-based mobile app built with **Flutter**, enabling users to control home devices, view energy usage, and monitor alerts â€” all in real-time with a sleek UI experience.

---

## Features

* Real-time device monitoring & control
* Smooth animations with glowing highlights
* Energy usage visual analytics
* Alerts & notifications section
* Clean bottom navigation (Dashboard | Energy | Alerts)
* Haptic feedback on toggles
* Modern dark-themed UI

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Muhammad-11Anas/Smart-Home-Dashboard-App

# Move to project directory
cd Smart-Home-Dashboard-App

# Get dependencies
flutter pub get

# Run the app
flutter run
```
